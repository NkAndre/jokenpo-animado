  ## JOKENPO COM ANIMAÇÕES !!   



## SOBRE O PROJETO  

Este projeto foi criado para praticar manipulação de DOM, eventos de tempo (setTimeout) e lógica de jogos simples em JavaScript.
O jogo simula uma disputa contra o computador, com animações de "balanço" antes de revelar o resultado.

## FUNCIONALIDADES

## Animações Sincronizadas:
As mãos balançam por 2 segundos antes da jogada ser revelada.

## Computador no modo random:
O computador escolhe uma opção de forma randômica.

## Interface Responsiva: 
Adaptado para diferentes tamanhos de tela.
